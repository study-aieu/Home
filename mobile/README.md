# Mobile (Expo)

Env (.env):

- EXPO_PUBLIC_API_BASE (e.g., http://localhost:4000)

Install and run:

- npm install
- npm start
- Press a for Android, i for iOS, w for web (Expo Go also supported)

Features:
- Login, Register
- Connect WordPress site (URL, Username, Application Password)
- List posts, Create/Update/Delete
- Markdown editing with preview
- Image upload to WordPress Media via backend
- Save drafts locally (SQLite)
